# Gold Price Prediction System

## 📌 Description

This project predicts gold prices using Random Forest, Gradient Boosting, LSTM, and Hybrid models.

## ⚙️ Features

* Data preprocessing
* Model training
* Evaluation (RMSE)

## 🚀 How to Run

1. Install dependencies
2. Run the main file

## 📊 Results

* Random Forest RMSE: 1112.45
* Gradient Boosting RMSE: 280.96
* LSTM RMSE: 3327.48
