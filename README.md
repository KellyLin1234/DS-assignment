# Gold Price Prediction Project

## Description
This project uses machine learning models (Random Forest, LSTM, Hybrid) to predict gold prices in 10 years.

## Models Used
- Random Forest
- Gradient Boosting / XGBoost
- LSTM
- Hybrid (RF + LSTM)

## Evaluation Metrics
- RMSE
- MAE
- R2
- MAPE

## Dataset
Gold Price.csv

## How to Run
Run the Python file to see results.
