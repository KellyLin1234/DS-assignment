# Gold Price Prediction (Hybrid Model)

## 📌 Project Description
This project predicts gold prices using a Hybrid model (LSTM + Random Forest).

## 📊 Models Used
- Random Forest
- Gradient Boosting / XGBoost
- LSTM
- Hybrid Model

## 📁 Dataset
Gold Price.csv

## ⚙️ How to Run
1. Install requirements
pip install -r requirements.txt

2. Run code
python main.py
